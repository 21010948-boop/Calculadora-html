# Calculadora-html

Aplicación web básica que permite realizar operaciones matemáticas simples.

## 🚀 Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## ⚙️ Funcionalidades
- Suma
- Resta
- Multiplicación
- División
- Interfaz interactiva

## 📂 Estructura del proyecto
- index.html
- styles.css
- script.js

## ▶️ Cómo ejecutar
1. Abrir el archivo `index.html` en el navegador
